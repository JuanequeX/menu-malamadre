# Menú Malamadre × Playas On Tech

Menú web de **Malamadre Plantas y Café** para el evento **Playas On Tech**, recreado en HTML/CSS a partir del diseño original.

- `index.html` — menú completo, responsive, sin dependencias de build.
- `assets/menu-original.png` — imagen original del menú.

## Deploy en Netlify

Es un sitio estático sin build:

1. En Netlify: **Add new site → Import an existing project**.
2. Conecta este repo de GitHub.
3. Deja **Build command** vacío y **Publish directory** en `/` (raíz).
4. Deploy.

También funciona arrastrando la carpeta al [Netlify Drop](https://app.netlify.com/drop).
